#  Weather App

A simple Python command-line application that fetches real-time weather information using the OpenWeather API.

## Features

* Search weather by city name
* Displays weather condition
* Shows temperature in Fahrenheit and Celsius
* Handles invalid city names

## Technologies

* Python
* Requests
* OpenWeather API

## Installation

```bash
pip install requests
```

## Usage

1. Add your OpenWeather API key to the script.
2. Run the program:

```bash
python weather.py
```

## Example

```text
Enter your city: Delhi

Weather: Clear Sky
Temperature: 91°F (33°C)
```
